Learn computing - Command Line, Shell, Terminal, Bash
Command lines can be typed through terminals. Then the operating system will run and execute the command.
Each command prompt is noted with “$” at the beginning. To find out the username, we can simply type: “whoami” into the terminal, and it will return the username.
Commands can also be combined by using the double ampersand (&&) and this allows the terminal to execute more than one command at a time.
To navigate to different files, the common line “cd” can change the directory.
To see the current directory, the command “pwd” can be used. To show list of files stored under the directory,
“ls” can be used. Adding “-la” to the the back of “ls” will also show the hidden files and the data
last modified.The terminal can also be used as a text editor through the installation of Vim and Nano.
Simply using “vim” at the start of the file will open the file in text editing mode. To edit, press “i” on the keyboard.
Afterward press “ESC” and “:wq” to save and exit the editor.
